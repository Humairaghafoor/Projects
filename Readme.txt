Setting up the development environment
There are a few technical requirements before we start.Install the following:
1-Node.js v8+ LTS and npm (comes with Node)
2-Git


Creating a Truffle project using a Truffle Box
1-  mkdir Project
2-  cd Project
3-  truffle unbox Project


Compiling and migrating the smart contract
1-  truffle compile
2-  truffle migrate
3-  truffle test


Using the dapp
npm run dev


Do not deploy the UI on Digital Ocean.Digital Ocean account requires some cost.
